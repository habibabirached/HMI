# Data Center Smart Power

A minimal React application for Data Center Smart Power.

## Getting Started

### Prerequisites
- Docker
- Docker Compose

### Running the Application

1. Build the Docker image:
```bash
./docker-dev-build.sh
```

2. Run the application:
```bash
./docker-dev-run.sh
```

The application will be available at http://localhost:3000/

3. Stop the application:
```bash
./docker-dev-stop.sh
```

## Development

The application is a simple React app that displays "Hello World". 

Source code is in `dcs-ui/src/` and is hot-reloaded when changes are made.
