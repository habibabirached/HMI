How the stack supports each M6 pillar

Multi-Personas / Multi-User — One API + one database mean the same truth for every role. PostgreSQL · FastAPI · Docker
Multi-Location / Multi-Device — URLs + real-time channels keep pan/zoom and state aligned. REST · WebSocket/SSE · Redux
Multi-Purpose — Simulation and other domains land in one pipeline and schema strategy. RTDS → PSCAD → FlexSIM → DB
Multi-Modal — Voice/gesture attach as a layer; optional cloud burst for heavy ML. NUI · Lambda (opt)
Multi-Dimensional — Consistent, rich plots everywhere. Plotly · shared API contracts
Governance — Reproducible releases and audit trail. Git · Jenkins · CI/CD → images
