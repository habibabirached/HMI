🔗 PSCAD Integration: Four Modalities

1

📁 CSV Upload

Engineers upload pre-computed simulation results through web interface. Perfect for demos and quick presentations. No PSCAD integration required.

2

🗄️ Database Connection

Direct connection to PSCAD result databases. Enables batch processing and automated scenario libraries with scheduled jobs.

3

🚀 API Integration

Real-time PSCAD execution via Python API and FlexSim. Interactive on-demand simulations responding to customer "what-if" questions instantly.

4

⚙️ Hardware-in-the-Loop

Live SCADA/DCS integration with OPC UA, Modbus, DNP3. Real-time operational monitoring, training, and digital twin capabilities.
